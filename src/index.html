Efficient Gesture Based Presentation Controller Using Transformer Learning Algorithm

Developed an AI-powered gesture-based system to control PowerPoint presentations using hand movements detected via webcam. The solution uses OpenCV, MediaPipe, and CNNs to recognize gestures for slide navigation, drawing, cursor movement, and undo operations. Designed to eliminate physical interaction with keyboard or mouse, this approach enhances presentation delivery. Achieved gesture recognition accuracy ranging from 80% to 100% under proper lighting conditions. The system supports real-time detection and offers a more interactive, hands-free presentation experience.
